# 100 Days of PowerShell

Welcome to my **100 Days of PowerShell** challenge, where I document my journey of learning PowerShell, building automation solutions, and developing skills that are directly applicable to modern IT environments.

This repository serves as both a learning journal and a portfolio, showcasing practical scripts, automation projects, and real-world solutions for IT professionals.

## The Goal

To develop practical PowerShell automation skills for:

* System Administration & Engineering
* IT Support & Help Desk Operations
* Microsoft 365 Administration
* Azure Administration & Cloud Computing
* Cybersecurity & Security Automation

By the end of this challenge, I aim to confidently use PowerShell to automate repetitive tasks, manage cloud resources, improve operational efficiency, and solve real-world IT problems.
