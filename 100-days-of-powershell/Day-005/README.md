# Day 005 - Network Connectivity Tester

## Objective

Develop a PowerShell script that performs basic network connectivity tests and generates a report.

## Features

* Tests connectivity to multiple hosts
* Measures average network latency
* Displays results in a formatted table
* Exports results to CSV

## Concepts Learned

* Test-Connection
* Arrays
* foreach loops
* PSCustomObject
* Measure-Object
* Export-Csv

## Real-World Use Case

Network connectivity testing is one of the first troubleshooting steps performed by IT Support technicians. This script automates basic diagnostics by checking whether important hosts are reachable and measuring response times.

## Skills Gained

* Basic network diagnostics
* Measuring latency
* Automating connectivity tests
* Building reusable troubleshooting tools

## Reflection

Today I built a PowerShell tool that performs automated network connectivity checks. It can quickly verify whether key hosts are reachable and provide latency information, making it useful for troubleshooting network-related issues.
